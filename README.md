# 発表時間管理 (Time App) v0.2.0

プレゼンテーションやLT（ライトニングトーク）の時間を管理するための、シンプルで強力なタイマーアプリです。

## 🚀 ダウンロード

**[最新バージョンのダウンロードはこちら (GitHub Releases)](https://github.com/mytakoyaki/time-app/releases/latest)**

*   **Windows**: `time-app_0.2.0_x64-setup.exe` (推奨) または `.msi`
*   **macOS**: `time-app_0.2.0_universal.dmg`
*   **Linux**: `time-app_0.2.0_amd64.AppImage` (インストール不要) または `.deb`

## ✨ 主な機能

*   **2画面プレゼン対応**: 外部ディスプレイに数字のみを全画面表示し、手元で操作。
*   **自動アップデート**: アプリ起動時に最新版への更新を通知。
*   **通知音プリセット**: 標準、電子音、ベル風、チャイム風から選択可能。
*   **柔軟な設定**: 発表と質疑応答の時間を個別に設定、超過分の自動差し引き。

## ⌨️ ショートカットキー

*   **Space**: 開始 / 一時停止
*   **Enter**: 次のステージへ進む（設定画面ではタイマー開始）
*   **R**: 現在のステージをリセット
*   **Esc**: 設定画面に戻る

## 🛠 インストール時の注意

### macOS
「開発元を検証できないため開けません」と表示される場合は、Finderでアプリを**右クリックして「開く」**を選択してください。

### Windows
青い画面（SmartScreen）が表示される場合は、「詳細情報」をクリックしてから「実行」を選択してください。
