# 発表時間管理アプリ (Time App) ユーザーガイド

このアプリケーションは、プレゼンテーションやLT（ライトニングトーク）の時間を管理するためのタイマーです。
発表時間と質疑応答時間を個別に設定し、残り時間が少なくなると色で警告します。

## 主な機能

*   **時間管理**: 発表時間と質疑応答時間を分・秒単位で設定できます。
*   **警告表示**: 設定した残り時間を切ると、文字色が黄色に変わります。
*   **超過表示**: 時間を超過すると文字色が赤に変わり、超過時間をカウントアップします。
*   **質疑応答の自動調整**: 発表が長引いた場合、超過した時間を次の質疑応答時間から自動的に差し引くことができます。
*   **プリセット機能**: よく使う時間設定を「プリセット」として保存・呼び出しできます。

## インストールと起動

（開発者向け: `npm run tauri build` で生成された実行ファイルを使用してください）

1.  アプリケーションアイコンをダブルクリックして起動します。

## 使い方

### 1. 時間の設定

起動直後の「設定画面」で以下の項目を設定します。

*   **発表時間**: プレゼンテーションの持ち時間。
*   **警告開始 (残り)**: 終了何秒前に黄色表示にするか（例: 60秒）。
*   **質疑応答**: 質疑応答の持ち時間。
*   **警告開始 (残り)**: 質疑応答の終了何秒前に黄色表示にするか。
*   **発表の超過分を質疑応答から引く**: 
    *   **オン (推奨)**: 発表が1分オーバーしたら、質疑応答がその分短くなります（全体の持ち時間を守るため）。
    *   **オフ**: 発表がオーバーしても、質疑応答は設定通りの時間で行われます。

### 2. タイマーの開始

「**タイマー開始**」ボタンを押すと、タイマー画面に切り替わり、自動的にカウントダウンが始まります。

*   **一時停止**: 「停止」ボタンを押すと止まります。「開始」で再開します。
*   **リセット**: 「リセット」ボタンを押すと、現在のステージ（発表なら発表）の時間を最初に戻します。

### 3. ステージの切り替え

発表が終わったら、「**次のステージへ**」ボタンを押します。

*   **自動開始**: ボタンを押すとすぐに次のステージ（質疑応答）のカウントダウンが始まります。
*   **時間調整**: 「発表の超過分を引く」設定がオンの場合、ここで計算が行われ、短縮された時間でスタートします。

### 4. 終了

全てのステージ（質疑応答まで）が終わると、「終了する」ボタンが表示されます。これを押すと設定画面に戻ります。

## プリセット機能

よく使う設定を保存しておけます。

### プリセットを使う
設定画面上部のボタン（例: 「標準」「LT」など）をクリックすると、その設定が入力欄に反映されます。そのまま「タイマー開始」を押せばすぐに始められます。

### プリセットを作る・編集する
1.  「**プリセットを管理・編集する**」リンクをクリックします。
2.  **新規作成**:
    *   「新規作成」タブを選びます。
    *   名前（例: 「社内会議」）と、各時間を入力します。
    *   「保存」を押すとリストに追加されます。
3.  **編集**:
    *   「保存済みリスト」タブで、編集したいプリセットをクリックします。
    *   数値を変更して「保存」を押します。
4.  **削除**:
    *   リストの右側にある「×」ボタンを押すと削除できます。

## 画面の見方

*   **白文字**: 通常運転中です。
*   **黄色文字**: もうすぐ終了です（警告）。
*   **赤文字**: 時間を超過しています（オーバータイム）。

---

## トラブルシューティング

*   **画面が真っ黒で数字しか見えない**:
    *   マウスを画面下部に移動させると、操作ボタンが表示されます。
*   **設定が保存されない**:
    *   アプリは設定を自動的に保存します。再起動してもプリセットは残ります。
